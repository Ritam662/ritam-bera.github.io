03 — Projects

01
American Football Training Platform
Full-Stack · Enterprise · USA Client

A full-stack sports training web application for American football with a multi-role ecosystem — Head Coach, Coach, Player, Fan, and Parent logins. Head coaches subscribe and onboard their full team. Built with Angular 15 (Cuba theme) on the frontend and ASP.NET controller-based API with PostgreSQL on the backend.

  • Role-based login system with subscription-gated access for Head Coach
  • Social media-style Feed page: posts, announcements, comments, replies
  • AWS ECS Fargate + Lambda integrations for cloud deployment
  • Multipart ZIP file uploads with S3 storage
  • Console application for background processing tasks
  • PostgreSQL stored procedures, functions & schema design
  • Entity Framework with scaffolding for data access layer
Angular 15 ASP.NET PostgreSQL AWS Fargate Lambda RBAC Cuba Theme
02
Microfinance Enterprise ERP
Full-Stack · Enterprise · Microservice Architecture

An enterprise-level microfinance management system — rebuilt and migrated to a modern stack. Led version migration from legacy architecture to latest Angular with Material Design and a microservices backend using ASP.NET Minimal API (endpoint-based). Deep SQL Server work including SSRS reporting, stored procedures, and advanced business logic.

  • Version migration: legacy system → modern architecture
  • Microservices with ASP.NET Minimal API (latest .NET architecture)
  • SQL Server + Microsoft SSRS for reporting & analytics
  • Background service + Console application for automated email sending
  • Advanced stored procedures and business logic layers
  • Angular Material UI with latest Angular version
  • Enterprise-scale data modeling and schema design
